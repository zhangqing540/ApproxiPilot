# Requirements

This directory contains the hardware and software environment information used in ApproxPilot experiments.

## Hardware

- GPU: NVIDIA P100 (32 GB)

## Software

- OS: Ubuntu
- CUDA: 11.7
- Python: 3.10
- PyTorch: 2.3.0
- PyTorch Geometric: 2.3.1
- torch-scatter: 2.1.1
- torch-sparse: 0.6.17

## Notes

The experimental code in this repository mainly relies on Python, PyTorch Geometric, and Synopsys Design Compiler (DC).
